# Tugas Mapid

## Cara Instalasi
### `Clone`
## Clone Article
```sh
git clone https://github.com/sanihamdhani/react-article.git
```
## Masuk Direktori
```sh
cd tugas-mapid
```
## Install Modul
```sh
npm install
```
## Install Axios
```sh
npm install axios
```
## Jalankan Projek
```sh
npm start
```
